# Nhánh - Brand

_Hiển thị danh sách nhánh trên local_

> **git brand**

_Tạo nhánh, ở yên tại nhánh_

> **git brand + tên nhánh**

_Tạo nhánh mới, xong chuyển qua nhánh đó luôn_

> **git checkout -b + tên nhánh**

_Đổi nhánh_

> **git checkout + tên nhánh**

_Hiển thị danh sách nhánh trên remote_

> **git branch -r**

_Câp nhật nhánh_

> **git fetch**
