#Hoàn tác cmt về trang thái ban đàu
_đưa thay đổi về change_

> **git reset + mã hard**

_đưa thay đổi về staged change_

> **git reset --soft + mã**

_mất luôn thay đổi_

> **git reset --hard + mã**

_mất thay đổi cần thiết_

> **git reset --merge + mã**

---

_sau khi xong, nếu muốn qua lại thì_

> **discard change + git pull**

_muốn cập nhật_

> **git push -f**
